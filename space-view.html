Add a live space view with javascript

Viewing our planet from the celestial heights is deeply beautiful and primordialy calming, for whatever reason. Maybe its the details of the clouds that spiral and weave with almighty foce. Even toddlers intrisically understand the power of weather. Maybe its the colors that scream out "home" The juxtaposition of water and sky wispers calm to our soul. Maybe its just bad-ass to be floating in space.

So lets see how to add this wonderful view using plain old vanilla javascript.

(image here)

The image above is a 20-minute-ago snapshot from the GEOS 17 satellite 22,200 mi above our planet. This is a weather satellite with a geostational orbit, which means it only has one static view of earth. The images are publically availible since 2018 at the NOAA site

There are quite a few cameras availible, but lets stick with the realistic "GeoColor" photo. After choosing a resolution we can get the URL of the image:

https://cdn.star.nesdis.noaa.gov/GOES17/ABI/FD/GEOCOLOR/5424x5424.jpg

Great, lets load it up.

    <img id="earth"></img>

(image here)

The image is extremely high resolution. Lets pretend we are orbiting the planet by slowly panning our view of the image.

#earth {
  animation-name: moveIt;
  animation-duration: 1000s;
  animation-timing-function: linear;
  animation-iteration-count: infinite;
}

@keyframes moveIt {
    0% {
        transform: translateX(-70%) translateY(-10%); 
    }
    25% { 
         transform: translateX(-0%) translateY(-10%); 
    }
    50% { 
         transform: translateX(-60%) translateY(-80%);   
    }
    75% { 
         transform: translateX(-0%) translateY(-50%); 
    }
    100% {
        transform: translateX(-70%) translateY(-10%); 
    }
}

There are several ways to achieve the panning effect. However not all of them use the GPU, causing jagged frame rate drop and lava-hot cpu temperatures.
